# My Portfolio

Premium personal portfolio for a B.Tech CSE (AI & ML) student, built with React, Vite, Tailwind CSS, Framer Motion, and React Three Fiber.

## Features

- Futuristic dark-first UI with glassmorphism panels and layered gradients
- Framer Motion powered section reveals, hover states, and loading screen
- React Three Fiber hero scene with an animated orbital object
- Interactive projects section with modal details
- Responsive layout optimized for desktop, tablet, and mobile
- Lazy loaded sections for better perceived performance
- Theme toggle with dark and light variants
- SEO-friendly metadata in `index.html`

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Three.js / React Three Fiber
- Lucide React

## Getting Started

1. Install Node.js 18+.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

## Customization

- Update personal content in `src/data/portfolioData.js`
- Replace placeholder project/demo/social links with real URLs
- Swap the hero/about visuals with personal assets if desired
- Adjust colors and glass effects in `src/index.css` and `tailwind.config.js`

## Deployment

This project is ready for Vercel or Netlify after installing dependencies and running a production build.
